使用 Hexo 搭建个人博客并部署到云服务器

1 整体流程

  1. 在本地运行 hexo deploy 命令，Hexo 会将生成的静态文件 (hexo generate) 推送到远程的 Git 仓库

  2. 在 Git 仓库中，配置一个钩子脚本，它会在接收到推送后执行——将最新的静态文件强制覆盖到指定的工作目录

  3. 在 Nginx 服务器中，配置访问根路径的请求指向该工作目录

  4. 本地环境准备
